国际 AI 资讯

计费规则调整与下一代模型节奏成为主线，Anthropic 重塑编程调用成本，OpenAI、Google 持续推进企业交付。

事项 · 计费调整时间 · 6月15日

Anthropic 把编程调用拆成独立额度：开发者成本重新计价。

自 6 月 15 日起，付费 Claude 计划中通过 Claude Agent SDK、claude -p、Claude Code GitHub Actions 以及第三方 Agent 产生的程序化用量，从订阅额度中拆出，转入单独的月度额度（Pro 20 美元、Max 5x 100 美元、Max 20x 200 美元），按完整 API 价格计量且不滚存。Claude.ai 网页对话与终端交互式会话不受影响。重度依赖自动化编程的团队需重新评估月度成本结构。

来源 · Anthropic 官方
事项 · 模型退役时间 · 6月15日

两款 Claude 4 模型从 API 退役

claude-sonnet-4-20250514 与 claude-opus-4-20250514 自 6 月 15 日从 Claude API 退役，此后对这两个模型的调用将返回错误。依赖旧模型 ID 的应用需尽快迁移到更新版本，避免线上服务中断。

来源 · Anthropic 模型退役文档
事项 · 企业生态时间 · 6月14日

OpenAI 推出 Partner Network，投入 1.5 亿美元

OpenAI 宣布推出 OpenAI Partner Network，投入约 1.5 亿美元资金，帮助全球合作伙伴加速企业 AI 的采用、部署与转型，覆盖咨询、系统集成与行业方案商，意在把模型能力更快转化为可交付的企业成果。

来源 · OpenAI 官方
事项 · 模型前瞻时间 · 6月13日

Gemini 3.5 Pro 仍处有限预览，GA 预计本月内

截至 6 月中旬，Gemini 3.5 Pro 仍处于 Vertex 有限预览与内部测试阶段，尚未正式 GA。该模型主打 200 万 token 超长上下文、“Deep Think” 深度推理模式与前沿多模态理解，将承接此前 Ultra 层处理的最难推理与长上下文任务，正式发布预计集中在 6 月下旬。

来源 · TechTimes
事项 · 企业平台时间 · 6月9日

微软联合 KPMG 全球规模化部署 AI 智能体

微软与 KPMG 深化合作，KPMG 通过 Microsoft Agent 365 在全球范围治理、部署与扩展可信企业级 AI 智能体，并在内部约 27.6 万名员工中规模化使用 Microsoft 365 Copilot，把智能体纳入审计、税务与咨询等业务流程。

事项 · 开源大模型时间 · 近日

DeepSeek V4 与 Kimi K2.6 同周发布万亿参数开源模型。

近期 Kimi K2.6 与 DeepSeek V4 两款万亿参数开源大模型在同一周内发布，目前均跻身 OpenRouter 全球调用量前列。DeepSeek 同步更新网页与 App 版本，支持最高 100 万 token 上下文，进一步缩小与海外旗舰模型的差距。国产开源路线在能力与生态上持续逼近第一梯队。
